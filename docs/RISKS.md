# Risk Register

## 1. Model accuracy is not good enough
Mitigation: sell triage/grade bands, build labeled dataset, focus on visible defect detection first.

## 2. Users submit bad photos
Mitigation: strict capture guidance and rejection. Make photo quality scoring core UX.

## 3. Legal/trademark issues
Mitigation: avoid implying affiliation with CGC/CBCS. Use “CGC-style” only carefully; prefer “professional grading estimate.”

## 4. No defensible data moat
Mitigation: opt-in dataset, partner with shops/pressers, blind submission feedback loop.

## 5. Existing competitors
Mitigation: differentiate through economics recommendations, batch workflows, pressing referral network, and transparent accuracy tracking.

## 6. Collectors distrust AI
Mitigation: position as second opinion before submission, not official grade. Let user inspect annotated evidence.
